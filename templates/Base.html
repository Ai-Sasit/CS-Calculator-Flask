Base Converter

This simple tool allows you to convert a same number between its representations under different number systems. For example, the number pi is equal to 3.14159... in a decimal system (base 10), or to 3.243F6... in a hexadecimal system (base 16), or to 11.001001... in a binary system (base 2).
